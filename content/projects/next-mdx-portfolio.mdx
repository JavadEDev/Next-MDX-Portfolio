# Next.js MDX Portfolio

This portfolio website is built with **Next.js** and **MDX**, allowing you to
showcase your projects and write blogs with ease. It combines the power of React
with the simplicity of Markdown, providing a great developer experience for
building dynamic and interactive pages.

## Technologies

This portfolio is powered by a stack of modern technologies:

- **Next.js**: A React framework for building server-side rendered and static
  websites.
- **MDX**: A combination of Markdown and JSX, allowing you to create interactive
  and dynamic content within your Markdown files.
- **React**: A JavaScript library for building user interfaces.
- **TypeScript**: (Optional) Provides static type checking to catch errors
  early.
- **Tailwind CSS**: A utility-first CSS framework for rapid styling and
  consistent design.
- **next-mdx-remote**: A library to handle MDX content in Next.js projects.
- **Gray-matter**: Parses front matter from MDX files to extract metadata like
  title, summary, and publication date.
